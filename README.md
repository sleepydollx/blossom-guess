# Blossom Guess

A cute theme terminal word guessing game, Hangman's softer cousin. Guess the secret word one letter at a time before the flower wilts away completely.

## How to play

1. Run the game (see below).
2. Pick a difficulty: easy, medium, or hard.
3. Guess letters one at a time.
4. Correct guesses fill in the word. Wrong guesses wilt the flower a little more.
5. Guess the full word before you run out of lives (6 total) to win!

## Running it

You'll need Python 3 installed. Then, from the project folder:

```bash
python blossom_project.py
```

or, depending on your system:

```bash
python3 blossom_project.py
```

No extra libraries needed — just plain Python.

## Difficulty levels

| Difficulty | Example words |
|------------|----------------|
| Easy       | pink, sweet, cloud, berry |
| Medium     | sparkle, blossom, ribbon, lullaby |
| Hard       | strawberry, watermelon, butterfly |
